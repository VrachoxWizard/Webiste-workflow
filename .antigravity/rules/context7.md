Use the `ctx7` CLI to fetch current documentation whenever the user asks about a library, framework, SDK, API, CLI tool, or cloud service.

1. Resolve library: `npx ctx7@latest library <name> "<user's question>"`
2. Fetch docs: `npx ctx7@latest docs <libraryId> "<user's question>"`

Use `--research` for deeper lookup if needed.
