```text

I am building an original premium ecommerce redesign inspired by pointershop.net.

This is not a 1:1 clone. The goal is to preserve the ecommerce structure and business type, but create a much better, modern, responsive, premium, natural, and less AI-generated website.

Stack:
- Next.js 16
- React
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Motion
- Lucide icons
- Playwright

Create a complete UX/UI and implementation plan.

Include:
1. sitemap
2. homepage layout
3. category listing layout
4. product detail layout
5. navbar and mega menu plan
6. mobile drawer menu plan
7. filter UX
8. cart UX
9. checkout mockup UX
10. visual design system
11. component architecture
12. mock data structure
13. responsible commerce/legal UX
14. anti-generic design rules
15. exact phase-by-phase implementation plan

Do not write code yet.
```
